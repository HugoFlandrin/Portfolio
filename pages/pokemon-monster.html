Pokemon Monster

A C# / WPF app inspired by the Pokémon battle system.

PokemonMonster is a desktop application built with WPF to recreate (in a simplified way) the classic Pokémon battle loop. The project includes authentication, catalogs for Pokémon and abilities, and a battle mode with progression to keep runs engaging.

Challenges

The focus was building a solid, readable app structure (MVVM) while keeping the battle logic simple, consistent, and fun.

  • Implementing MVVM while keeping navigation practical
  • Connecting and initializing the database reliably (SQL Server)
  • Balancing “simple rules” with meaningful progression

What I built

  • Login / registration flow
  • Catalog of the first 151 Pokémon (stats + abilities)
  • Abilities catalog (details + compatible Pokémon)
  • Battle system with dynamic difficulty after 3 victories
  • Top navigation bar to switch between key pages

Technologies used

  • C#
  • WPF
  • SQL Server
  • MVVM
